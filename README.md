# 🛒 Product Catalog API

A RESTful API for managing product data with full CRUD operations using **Node.js**, **Express**, and **MongoDB (Mongoose)**.

---

## 🚀 Features

- Create a new product
- Get all products
- Get a single product by ID
- Update product details
- Delete a product
- Error handling and JSON responses

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
